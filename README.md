# LogicFlow Vue Demo

## 技术栈：
* LogicFlow + Ant Design Vue
* Vue3 + Vite + TS

## 开发环境：
本项目使用 pnpm 管理依赖，使用 vite 进行构建打包。

一般步骤：

1. 下载依赖包

   ``` shell
   pnpm i
   ```

2. 开发运行

   ``` shell
   pnpm dev
   ```

3. 编译打包

   ``` shell
   pnpm build
   # 或 清理后重新打包
   pnpm all
   ```

4. 运行打包版本

   ``` shell
   pnpm preview
   ```
